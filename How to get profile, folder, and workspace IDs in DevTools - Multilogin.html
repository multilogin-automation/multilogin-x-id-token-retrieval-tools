How to get profile, folder, and workspace IDs in DevTools

Written by Yelena V ( Updated on March 27th, 2026 )

Updated on March 27th, 2026

Profile, folder, and workspace IDs are required to run the majority of API requests with Multilogin. Let's explore how to get them using Developer Tools.

How to get profile IDs

It is easy to find profile IDs in Multilogin interface:

  1. Select a profile or several profiles in the list using checkboxes on the left
  2. Click “Copy profile ID” in the top menu

How to get workspace and folder IDs

Multilogin comes as a desktop app and has a web-accessible interface. Pick the version that works best for you!

Desktop app

  1. Open Multilogin app
  2. Open DevTools. Here's how to do that:
    1. Windows and Linux: press Ctrl + Shift + F12
    2. macOS: press Cmd + Shift + Fn + F12
  3. Switch to the “Network” tab in the DevTools
  4. Click the Multilogin logo at the top left of the sidebar
  5. A dropdown will appear with all the workspaces you’re part of
  6. Click the one you want to switch to
  7. In the logs appearing in DevTools, search for the word user or folders and click on it
  8. Switch to the “Preview” tab in the newly opened panel, where you will find the workspace ID or folder

Refer to the How to switch between Multilogin workspaces article for more details about switching workspaces.

 
 
 

Web interface

  1. Go to the “Profiles” page in Multilogin.
  2. Open DevTools. Here's how to do that for Chromium- and Firefox-based browsers:
    1. Windows and Linux: press Ctrl + Shift + I
    2. macOS: press Cmd + Option + I
  3. Switch to the “Network” tab in the DevTools panel and refresh the page
  4. In the logs appearing in DevTools, search for the word workspaces or folders and click on it
  5. Switch to the “Response” tab in the newly opened panel, where you will find the workspace IDs
 
 

How to get workspace (and default folder) ID via GUI

You can get workspace and default folder ID via GUI. The workspace ID is the same as the ID of the default folder. If you don't need the other folders – you can follow the steps below:

  1. Click the Multilogin logo in the upper left corner and select the workspace you need
  2. Click “Account” → “Account settings” in the left bottom corner
  3. Click the copy button next to the “Workspace ID” field
